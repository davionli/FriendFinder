# FriendFinder

A [website](https://davion-friend-match.herokuapp.com/) to match friends by anwsering some questions.


## Overview

Click `Match Now` button to start your survey.

After you answring all ten questions, the best matched people will be displayed.

You can check the existing person by click `API Friends List` on the bottom.

## Support

* Node.js

* express
